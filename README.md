# AI Resume Analyzer 📄🤖

An AI-powered Resume Analyzer built with Streamlit that helps job seekers evaluate their resumes against job descriptions.

## Features

* Upload PDF resumes
* Extract skills automatically
* Calculate Resume Match Score
* Calculate ATS Score
* Identify Missing Skills
* Generate AI-based feedback
* Visualize skills with charts
* Download PDF analysis reports

## Technologies Used

* Python
* Streamlit
* Scikit-Learn
* PyPDF2
* Matplotlib
* ReportLab

## How It Works

1. Upload your resume (PDF)
2. Paste a job description
3. Analyze resume skills
4. View ATS and Match Scores
5. Download a PDF report

## Installation

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app.py
```
